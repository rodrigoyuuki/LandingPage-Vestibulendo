Projeto: Vestibulendo
Descrição: O site tem como objetivo divulgar o aplicativo Vestibulendo, apresentando tópicos como a Problemática, Justificativa e o Plano Premium no app. 
O vestibulendo é uma plataforma literária preparatória para os vestibulares paulistas, assim, traz como público-alvo os estudantes e vestibulandos. 
A landing page utiliza as cores do projeto e seus elementos visuais como as logos.
